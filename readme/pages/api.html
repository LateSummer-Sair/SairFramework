框架核心 API 参考

SairFramework 核心类的完整 API 说明，插件开发者需要了解的关键接口。

Main 类 sair.Main

框架的入口类，负责启动流程和插件测试。

成员类型说明
Versionstatic String当前框架版本号（如 "0.5.3"）
toTest(Activity, String, String)static Object调试用：加载框架并测试指定 Activity 的指定功能

Activity 抽象类 sair.user.Activity

所有插件的基类，开发者必须继承此类。

方法说明
main(String funcName, String args)抽象方法，处理命令的主入口
help()抽象方法，返回帮助字符串数组
exit()抽象方法，插件卸载时调用
dataDir()可重写，返回自定义数据目录名，默认返回 null（使用类全名）
o_funcMain(Object o)可重写，处理 ofunc 命令的返回值，默认打印错误信息
getDataDir()获取数据目录的完整路径，自动创建目录
getName()获取插件名称
setName(String)设置插件名称（名称不能重复）
isOpen()检查插件是否处于打开状态
close()关闭插件命令接收
open()打开插件命令接收
putOrderName(newName, funcName)创建命令别名
getOldOrderName(newName)获取别名对应的原始命令名
removeOrderName(name)删除命令别名
containsOrderName(name)检查是否存在指定别名

SairCons 类 sair.sys.SairCons

控制台操作核心类，提供打印输出、命令执行等功能。

方法说明
print(String info)使用默认颜色打印文本
print(Color c, String info)使用指定颜色打印文本
println(String info)使用默认颜色打印文本并换行
println(Color c, String info)使用指定颜色打印文本并换行
clear()清空控制台内容
getConsoleText()获取控制台所有文本
getConsoleSize()获取控制台内容大小
dePrint(Integer offs, Integer len)删除指定范围的内容
insertPrinto(Integer index, Color c, String info)在指定位置插入文本（考虑代理模式）
runner(boolean isMark, String cmd)执行命令字符串（isMark=true 则记录历史）
toActiRun(Activity, String funcName, String args)直接调用 Activity 的 main 方法
getDefaultColor()获取默认字体颜色
setDefaultColor(Color)设置默认字体颜色
addPrintRunnable(String id, PrintRunnable)添加打印代理（输出拦截器）
removePrintRunnable(String id)移除打印代理
removeAllPrintRunnable()移除所有打印代理
SpliterSpiManagerstatic SpliterSPI — 当前活跃的命令解析器
localRunnerHistorystatic ArrayList<String> — 命令历史记录

Libraries 类 sair.sys.Libraries

全局注册表，管理所有已加载的 Activity、Exection 和 Mod。

成员类型说明
activitiesHashMap<String, Activity>按名称索引的所有 Activity
exectionsHashMap<Activity, Exection>Activity 到 Exection 的映射
modsHashMap<String, Mod>所有已加载的 Mod 库
setActivityName(String path)static String根据 JAR 路径生成唯一 Activity 名称

LoaderManager 类 sair.LoaderManager

类加载管理器，负责加载 bootlib、modlib 和 exection 中的 JAR。

成员说明
systemLoader系统 ClassLoader
loader全局 SairLoader 实例（用于 modlib）
ExecLoadersHashMap<URL, SairLoader> — 每个插件的独立 ClassLoader 映射
loadExecJar(String)加载一个插件 JAR 到独立 ExectionLoader
loadLibJar(String)加载一个库 JAR 到全局 SairLoader
loadMain(String className, ClassLoader)从指定 ClassLoader 加载并实例化 Activity
getModResStream(String)从 modlib 的 JAR 中读取资源文件流

ToolPack 类 sair.sys.tools.ToolPack

工具集，提供路径解析、MANIFEST 读取、命令分割等功能。

方法说明
getPath()获取 SFW.jar 所在目录的绝对路径
getAllFilesPath(File, boolean)递归获取目录下所有文件的路径列表
pathRepack(String)解析带双引号的路径字符串，返回绝对路径数组
getExeMain(String)读取 JAR 中 MANIFEST.MF 的 ACT 属性
getAttributes(String name, String path)读取 JAR 中 MANIFEST.MF 的指定属性
findSplited(String cmd)获取当前命令解析器解析命令
setSpliter(String className)切换命令解析器
IntegerValOfString(String)从字符串中提取数字
getVmap()获取环境变量表 HashMap<String,String>
reArg(String[], Integer[])从参数数组中排除指定索引的元素后重组
chkArgsV(String)替换参数中的 %变量名% 为实际值

Pathes 类 sair.Pathes

路径常量定义类。

成员说明
bootDirplugins/bootlib/ 完整路径
modDirplugins/modlib/ 完整路径
execDirplugins/exection/ 完整路径
dataResDirdata/ 完整路径
printSplit分割线字符串 "----------..."
logoPathLogo 图片路径
fontPath字体文件路径

FCM 类 sair.FCM

框架颜色管理（Framework Color Manager），定义各场景的默认颜色。

成员默认值说明
loadBoot_ColorBLACKbootlib 加载日志颜色
loadExection_ColorORANGEexection 加载日志颜色
loadMod_ColorWHITEmodlib 加载日志颜色
EXECTION_pathInfo_ColorYELLOWinfo 命令输出的路径信息颜色
EXECTION_help_ColorPINKhelp 命令输出的帮助信息颜色
Error_ColorRED错误信息颜色
split_ColorGREEN分割线颜色

Spliter 接口 sair.sys.tools.Spliter

命令分割器接口。默认实现为 SystemSpliter。

方法说明
getExecName()获取 Activity 名称（命令中 / 之前的部分）
getExecFunc()获取功能名称（/ 之后、空格之前）
getArgs()获取参数字符串（空格之后全部内容）

PrintRunnable 接口 sair.user.PrintRunnable

打印代理接口。实现此接口可以拦截控制台输出。

public interface PrintRunnable {
    void run(Integer index, Color c, String info);
}

注册方式：SairCons.addPrintRunnable("myPrinter", myPrintRunnable);
